# FOC-Linear-Motor : FOC控制線性電機
FOC Linear Motor 是一個微型的直線電動機。

## 概要
1. 體積小巧 : 58*20*23mm
2. 移動距離 : 20mm
3. 電源輸入 : 5V/2A
4. MCU : STM32F405
5. 帶有電流感測、電壓感測、中心抽頭感測
6. 基於條型編碼器和End-Stop估測位置
